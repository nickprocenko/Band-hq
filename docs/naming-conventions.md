# Naming conventions (Citrus тАУ HQ)

## General
- Prefer short, consistent names over "final" / "final-final".
- Use versions when needed: v1, v2, v3.
- Use dashes to separate parts: `YYYY-MM-DD тАУ Thing тАУ Details`.

## Dates
- For anything date-based, start with `YYYY-MM-DD`.
  - Example: `2026-04-10 тАУ Rehearsal notes`

## Covers vs originals
- Covers: `Artist тАУ Song Title`
- Originals: `Song Title`

## Audio exports
- `Song Title тАУ demo тАУ YYYY-MM-DD.mp3`
- `Song Title тАУ mix v3 тАУ YYYY-MM-DD.wav`

## Charts/lyrics
- `Song Title тАУ Lyrics`
- `Song Title тАУ Chords`
- `Song Title тАУ Chart`
