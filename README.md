# Band HQ

A lightweight, shared “operating system” for organizing a band’s files (Google Drive-first), with a structure that can later become an app.

## Band
- Name: Citrus
- Members: Johnny, Damian, Sam, Nick, Matt

## What this repo contains
- `docs/`: the rules (folder structure, naming, workflows)
- `templates/`: copy/paste templates for docs and sheet headers
- `citrus/`: Citrus-specific snapshot (members list + drive tree)

## Quick start
1. Create a shared Google Drive folder: `Citrus – HQ`
2. Apply the structure in `docs/drive-structure.md`
3. Use templates in `templates/` for member tracking and suggested songs
